# Consistency Checklist

Use this checklist when updating docs/wiki content.

## Accuracy checks

- Commands in wiki pages match `README.md` and executable scripts.
- Output file names/paths match current code behavior:
  - `figures/simulation-histogram-generated.png` (generated by `simulation.py`)
  - `data/simulation-results.csv` (generated by `src/simulate.py`)
  - `data/skir-control-results.json` (generated by `tools/run_simulation_controls.py`)
- Validation commands match `CONTRIBUTING.md`.
- LaTeX compilation uses `latex/references.bib`.
- CI workflow paths and tool references match `.github/workflows/ci.yml` and `.github/workflows/skir-validation.yml`.
- Claim language follows `docs/claim-language-policy.md`.
