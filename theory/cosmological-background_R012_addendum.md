# R-012 Addendum: Cosmological Background Equation

R-012 adds:

\[
E_{\mathrm{ASH}}^2(a)=
\Omega_r a^{-4}+\Omega_m a^{-3}+\Omega_k a^{-2}+\Omega_\Lambda
+\Omega_{\mathrm{ASH}}\Xi_{\mathrm{ASH}}(a).
\]

The source is a normalized finite-observer workbench source:

\[
\Xi_{\mathrm{ASH}}(a)=
\frac{\alpha S_b(a)+\beta D_b(a)+\gamma M_b(a)}
{\alpha+\beta+\gamma}.
\]

Setting \(\Omega_{\mathrm{ASH}}=0\) exactly recovers the standard baseline expression. This is not a continuum gravity derivation.
