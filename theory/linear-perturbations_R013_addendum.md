# R-013 Addendum to Linear Perturbations

R-013 extends the finite R-007 perturbation sector with a bounded physical \(k\)-bearing scalar matter-growth solver.

It uses the finite spectral table \((\lambda_q,m_q)=(0,1),(16,9),(28,36),(36,84),(40,126)\) and maps it into the versioned kernel \(K_{\rm ASH}(k)\).

The solver is Newtonian-gauge, scalar-sector, sub-horizon/quasi-static. It is not a gauge-invariant perturbation theory, photon-baryon Boltzmann hierarchy, tensor/vector solver, or empirical result.
