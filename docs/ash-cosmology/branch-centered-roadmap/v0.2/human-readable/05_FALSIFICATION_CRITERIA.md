# Falsification Criteria

ASH Cosmology should be rejected, revised, or demoted if any of the following occur.

| ID | Failure | Consequence |
|---|---|---|
| F0 | finite local algebra contradiction | reject affected finite theorem |
| F1 | branch tree cannot be defined | no leaf-world cosmology |
| F2 | branch weights or amplitudes cannot normalize | no probabilistic/quantum branch interpretation |
| F3 | Born-rule or replacement frequency theorem cannot be derived | no Everett-style ASH claim |
| F4 | no decoherence or branch separation mechanism | leaves are not world-like |
| F5 | observer memories are not embedded in branches | commitment model fails |
| F6 | no branch-to-geometry bridge | no cosmology claim |
| F7 | no stable continuum or causal limit | no standard spacetime claim |
| F8 | matched controls reproduce signatures | effect not ASH-specific |
| F9 | observables cannot be computed from fixed equations | no empirical validation |
| F10 | predictions are revised after unblinding | empirical claim invalid |

Falsification is a feature, not a defect. It is what turns ASH from interpretation into science.
