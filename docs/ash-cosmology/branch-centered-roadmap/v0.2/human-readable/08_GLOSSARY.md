# Glossary

| Term | Definition |
|---|---|
| ASH | Adinkra-Stabilized Hypercube / Enneahcube information-cosmology framework. |
| Enneahcube | 9-bit hypercube over \(\mathbb F_2^9\). |
| finite local layer | Local state alphabet, code, parity, decoder, and defect policy. |
| leaf branch | Frontier history in the ASH branch tree. |
| branch tree | Unbounded family of admissible possible histories. |
| commitment | Observer-relative realized record; not necessarily collapse. |
| branch measure | Normalized nonnegative weight over leaves. |
| branch amplitude | Complex amplitude over leaves for quantum-like ASH. |
| decoherence functional | Rule measuring branch separation/noninterference. |
| branch entropy | Entropy of leaf weights or amplitudes over a frontier. |
| bridge map | Map from branch ensemble to effective physical observables. |
| matched control | Non-ASH system matched to isolate whether an effect is ASH-specific. |
