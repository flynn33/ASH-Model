# Branch-Centered Correction Lock

## Locked correction

ASH Cosmology must not be framed as merely finite. The finite layer is the local alphabet. The leaf branches are the cosmological book of possible worlds.

```text
finite local state → infinite branch histories → branch amplitudes/measures → decoherence → observer-relative commitment → emergent geometry → cosmological observables
```

## What changed

The earlier roadmap placed branch calculus after finite proof closure. That made branching look secondary. The corrected model puts branch histories inside the primary object:

\[
\mathfrak A=(\Omega,C,D,\Gamma,\mathcal T,\mu/\psi,\mathcal C,\mathcal M,\mathcal B_\ell).
\]

## What must be protected

1. The finite Enneahcube/kernel is local, not total.
2. Leaf branches are first-class cosmological objects.
3. Commitment is observer-relative and need not collapse the branch tree.
4. A many-worlds ASH claim requires branch ontology, amplitude/measure law, decoherence, and a Born-rule or replacement theorem.
5. The physical bridge must map branch ensembles, not finite states alone.
