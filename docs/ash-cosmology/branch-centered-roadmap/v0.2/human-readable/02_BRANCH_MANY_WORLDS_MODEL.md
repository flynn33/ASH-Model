# ASH Leaf-Branch Many-Worlds Model

## Branch interpretations

ASH leaf branches can be interpreted in three ways:

| Interpretation | Meaning | Cosmology strength | Required structure |
|---|---|---|---|
| Epistemic | possible reconstructions or hypotheses | weak | scoring and pruning |
| Ontic non-quantum | physically real histories | strong | branch ontology, measure, conservation law |
| Quantum/Everett | decohered components of universal state | strongest | Hilbert space, amplitudes, norm preservation, decoherence, Born rule or alternative |

For ASH Cosmology, the serious candidates are:

1. ontic non-quantum branches with explicit measure \(\mu\); or
2. quantum/Everett branches with complex amplitudes \(\psi\).

## Deterministic branching

All admissible children exist:

\[
\mathcal F_{t+1}=\operatorname{expand}(\mathcal F_t).
\]

## Weighted branching

Each child has nonnegative weight:

\[
w(b')\ge0, \quad \sum_{b'\in children(b)} w(b') = w(b).
\]

## Quantum-like branching

Each child has amplitude:

\[
\psi(b')\in\mathbb C, \quad \sum_{b'} |\psi(b')|^2 = |\psi(b)|^2.
\]

The universal branch state can be written:

\[
\Psi_t = \sum_{\ell\in L_t} \psi_\ell |\ellangle.
\]

If ASH claims a many-worlds interpretation, it must solve or explicitly replace the Born-rule problem:

\[
P(\ell)=|\psi_\ell|^2.
\]

## Commitment

Commitment should be observer-relative, not necessarily collapse:

```text
An observer-history records one branch while the full branch tree remains mathematically real.
```

Let observer memory be:

\[
M_O(t)\subset b.
\]

Observation locates an observer on a leaf; it does not automatically delete other leaves.
