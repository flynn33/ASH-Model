# 1. Research objects and canonical definitions

## 1.1 Enneahcube state space

Define the finite state kernel:

\[
\Omega = \mathbb F_2^9.
\]

Each state is

\[
x=(x_1,\ldots,x_9),
\qquad x_i\in\{0,1\}.
\]

The Enneahcube graph has one vertex for each state and one edge for every one-bit flip:

\[
x \sim y
\iff
d_H(x,y)=1.
\]

Immediate theorem targets:

| Object | Required proof |
|---|---|
| State count | \(|\Omega|=2^9=512\) |
| Vertex degree | every vertex has 9 neighbors |
| Hamming planes | \(P_k=\{x:\mathrm{wt}(x)=k\}\) |
| Plane count | \(|P_k|=\binom{9}{k}\) |
| Bit-flip graph | regular, bipartite, connected |

## 1.2 ASH code layer

Define a canonical rank-4 doubly-even linear code:

\[
C \subset \mathbb F_2^9,
\qquad
[n,k,d]=[9,4,4].
\]

Required theorem targets:

| Claim | Proof target |
|---|---|
| Linearity | \(C\) is closed under XOR |
| Rank | \(\dim C=4\) |
| Size | \(|C|=16\) |
| Doubly-even | \(\mathrm{wt}(c)\equiv0\pmod4\) for all \(c\in C\) |
| Minimum distance | \(d_{\min}=4\) |
| Single-error correction | unique correction for \(t=1\) |
| Double-error policy | detect/reject, not silently heal |
| Not self-dual | \(\dim C\neq 9/2\) |

For ASH Cosmology, the code cannot remain merely a convenient error-correcting primitive. The roadmap must determine whether \(C\) functions as:

- a gauge constraint;
- an admissibility constraint;
- a local conservation law;
- a causal update alphabet;
- a parity/integrity condition;
- a stabilizing subspace;
- a microphysical symmetry.

Those are different scientific claims. We must choose and prove the consequences.

## 1.3 Dual-mode branch structure

Define ASH as:

\[
\mathcal A = (\Omega,C,D,G,B,S,P,M).
\]

| Symbol | Meaning |
|---|---|
| \(\Omega\) | finite state space |
| \(C\) | code / admissibility / correction structure |
| \(D\) | decoder and defect policy |
| \(G\) | recursive branch grammar |
| \(B\) | branch expansion operators |
| \(S\) | scoring or action functional |
| \(P\) | pruning / commitment policy |
| \(M\) | realized memory / history graph |

The branch layer is not finite in principle. At finite runtime, only a bounded frontier is explored; mathematically, the tree can be unbounded.

Define a branch tree:

\[
\mathcal T_x
=
\{b: b \text{ is a finite branch path rooted at } x\}.
\]

Each branch has:

\[
\mathrm{state}(b)\in\Omega,
\quad
\mathrm{weight}(b)\in\mathbb R_{\ge0},
\quad
\mathrm{score}(b)\in\mathbb R,
\quad
\mathrm{parent}(b),
\quad
\mathrm{children}(b).
\]

Commitment selects finite realized output:

\[
\operatorname{commit}
:
\mathcal F_t
\to
\Omega
\quad\text{or}\quad
\operatorname{commit}
:
\mathcal F_t
\to
\text{observable event},
\]

where \(\mathcal F_t\) is the explored frontier at step \(t\).

Required theorem targets:

| Claim | Required proof |
|---|---|
| Branch closure | child states remain admissible or are explicitly rejected |
| Weight normalization | branch weights sum to 1 when interpreted probabilistically |
| Commit determinism | same inputs produce same committed output if deterministic |
| Commit stability | small perturbations do not cause pathological branch jumps |
| Pruning safety | pruning cannot remove all admissible branches unless failure is declared |
| Infinite limit | if depth \(\to\infty\), the branch measure is well-defined or deliberately non-probabilistic |

# 2. ASH Cosmology theory target

The central object must be a universe model, not a single-state model.

## 2.1 Microscopic universe state

Let \(\Gamma\) be a causal graph, lattice, hypergraph, or event network.

A microscopic ASH universe at update index \(t\) is:

\[
X_t
=
\{x_v(t): v\in \Gamma_t,\ x_v(t)\in\Omega\}.
\]

If the graph itself evolves:

\[
\mathcal U_t=(\Gamma_t,X_t).
\]

Research decision required:

| Option | Meaning |
|---|---|
| Fixed graph | ASH states live on a pre-existing causal lattice |
| Dynamic graph | ASH states generate or modify adjacency |
| Branching graph | many possible graph futures exist, then commit |
| Emergent graph | adjacency is derived from state correlations |

The most powerful cosmology path is dynamic or emergent graph structure, but the most mathematically tractable first path is fixed causal graph.

## 2.2 Local update law

Define a local transition rule:

\[
x_v(t+1)
=
F_\theta
\left(
x_v(t),
\{x_u(t):u\in N(v)\},
\eta_v(t)
\right).
\]

Or stochastic form:

\[
K_\theta
\left(
X_{t+1}\mid X_t
\right).
\]

Or branch form:

\[
\mathcal F_{t+1}
=
\operatorname{expand}
\left(
\mathcal F_t;G,B,S
\right),
\qquad
X_{t+1}
=
\operatorname{commit}(\mathcal F_{t+1};P).
\]

Mathematical gates:

1. \(K_\theta\) must be normalized.
2. Invalid states must be impossible or handled.
3. Causal influence must be bounded.
4. Conservation laws must be explicit.
5. Parameters \(\theta\) must be finite and interpretable.
6. The rule must not be retrofitted after data.

## 2.3 Physical bridge map

ASH becomes cosmology only when there is a map:

\[
\mathcal B_\ell:
(\Gamma,X)
\longrightarrow
(g_{\mu\nu},T_{\mu\nu},\phi^A,J^\mu,\rho,p,\ldots)_\ell.
\]

This is the most important missing theory object.

It must answer:

| ASH object | Physical interpretation needed |
|---|---|
| Bit coordinate | field component, constraint, charge, mode, or symbolic state |
| Codeword | gauge move, transition generator, correction mask, or local symmetry |
| Hamming distance | physical distance, action cost, defect measure, or none |
| Branch | possible history, computational hypothesis, quantum-like alternative, or reconstruction candidate |
| Commit | measurement, decoherence, coarse-graining, classicalization, or inference |
| Memory graph | spacetime history, causal set, path integral support, or observer record |

Until \(\mathcal B_\ell\) exists, ASH Cosmology has no measurable observables.

# 8. Candidate mathematical research questions

These are the central problems I would attack first.

## Q1. Can code cosets define conserved sectors?

Study whether ASH code cosets behave like superselection sectors:

\[
\Omega / C.
\]

If yes, prove what transitions are allowed and forbidden.

## Q2. Can branch depth produce a natural action?

Try:

\[
S(b)
=
\alpha\,d_H(x_b,x_0)
+
\beta\,\ell(b)
+
\gamma\,\mathrm{defect}(b)
+
\delta\,\mathrm{curvature}(b).
\]

Then define:

\[
P(b)
=
\frac{e^{-S(b)}}{Z}.
\]

This may become a bridge between branch scoring and statistical physics.

## Q3. Can Enneahcube spectra create observable scales?

The 9-cube adjacency spectrum is highly structured. Study whether code-projected spectra produce fixed eigenvalue families that could map to physical oscillation or damping modes.

## Q4. Can parity/integrity defects behave like particles?

Define a defect:

\[
\epsilon(x)=x_9\oplus(x_1\oplus\cdots\oplus x_8).
\]

Study defect propagation, annihilation, conservation, and interaction.

## Q5. Can committed branch histories generate emergent time?

Let \(M_t\) be committed branch memory. Investigate whether causal order arises from:

\[
M_0 \subset M_1 \subset M_2 \subset \cdots.
\]

This may produce an arrow of time from commitment, not from the finite kernel alone.

## Q6. Can branch entropy become cosmological entropy?

Define:

\[
S_{\rm branch}(t)
=
-\sum_{b\in\mathcal F_t}p_b\log p_b.
\]

Then test whether committed histories obey entropy growth or apparent horizon-like relations.
