# ASH Cosmology Branch-Centered Roadmap Package

**Package version:** 0.2.0-branch-centered-roadmap
**Scope:** research, mathematics, empirical evidence, falsification, and documentation of ASH Cosmology.
**Explicitly excluded:** repository management, storage advice, implementation workflow, governance, and file-placement decisions.

## Locked correction

ASH Cosmology must not be described as merely finite. The finite Enneahcube/code layer is the **local integrity alphabet**. The leaf-branch structure is a first-class cosmological object and the candidate many-worlds / branching-history layer.

```text
ASH Cosmology = finite local Enneahcube state kernel
              + recursive leaf-branch universe
              + branch measure / amplitude / score law
              + commitment / observation rule
              + physical bridge to spacetime and observables
```

## Package contents

- `human_readable/` contains the research roadmap, canonical model, branch many-worlds framework, proof program, empirical validation plan, falsification criteria, prediction-ledger guide, and glossary.
- `machine_readable/` contains JSON/YAML manifests, schemas, theorem catalogs, phase catalogs, experiment catalogs, falsification rules, prediction templates, and dependency graph.
- `tables/` contains CSV summaries for phases, experiments, and falsification rules.
- `source_evidence/` contains copied source evidence files used to anchor this roadmap. These files are included as evidence, not as repository-management guidance.

## One-line roadmap

```text
Enneahcube local states → admissible branch histories → branch measure/amplitude law → decoherence and observer-relative commitment → branch-ensemble geometry → cosmological observables → empirical tests.
```
