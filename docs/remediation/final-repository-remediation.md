# Final Repository Remediation Evidence

This directory records the last repository-remediation pass before ASH project work moves into science.

## Required evidence files

- `final-remediation-evidence.json`
- `physics-readiness.json`

## Required interpretation

A passing repository-remediation audit means:

- finite ASH kernel checks pass;
- generated artifacts are current;
- manuscript verification policy is enforced;
- JSON schemas and instances agree;
- repository gates are aligned;
- project policy guards remain active;
- science blockers are visible and not mislabeled as complete.

A passing repository-remediation audit does **not** mean:

- physical state ontology has been chosen;
- microscopic dynamics have been derived;
- bridge maps to observables have been defined;
- cosmological equations have been solved;
- empirical predictions have been tested.

Those belong to the science phase.
