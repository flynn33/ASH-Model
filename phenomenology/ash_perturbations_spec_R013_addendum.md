# R-013 Addendum to ASH Perturbation Observable Specification

R-013 emits:

- `r013_matter_power_proxy.csv`
- `r013_cmb_lowell_proxy.csv`
- `r013_finite_to_physical_modes.csv`
- `r013_synthetic_matter_observations.csv`
- `r013_alpha_mu_grid_fit.csv`

`P_proxy` is a dimensionless transfer-amplitude proxy, not a calibrated survey \(P(k)\). The CMB file is a low-ell interface proxy only.

No external data, covariance matrix, or likelihood is used. R-014 remains responsible for empirical validation infrastructure.
