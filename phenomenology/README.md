# Phenomenology Specifications

Status: Draft

This directory defines the future observable interface for ASH-Physics validation. It does not claim that ASH has passed observational tests.

## Files

- `ash_background_spec.md`: background-variable and equation requirements.
- `ash_perturbations_spec.md`: perturbation-variable and mode-evolution requirements.
- `primordial_spectrum_spec.md`: requirements for any primordial-spectrum prediction.
- `observables_spec.md`: mapping from model outputs to measurable statistics.

## Boundary

Phenomenology work remains blocked until theory files define physical state variables, dynamics, a bridge map, and a baseline limit.
